Fizz Buzz Checker

Params

Number

Return

Return Fizz if number divisible by 2

Return Buzz if number divisble by 3

Return Fizz Buzz if number divisible 2 and 3

Example

[2], return 'Fizz'

[3], return 'Buzz'

[15], return 'Fizz Buzz'

Psuedo

if (% 2 === 0){ console log 'Fizz'}

if (% 3 === 0){ console log 'Buzz'}

if (% 15 === 0){ console log 'Fizz Buzz'}
